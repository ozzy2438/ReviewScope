About This Tool

This web application allows you to search for products on Amazon and perform comprehensive analyses on the results:

  • Searches for the products you specify on Amazon
  • Collects price, rating, and review counts
  • Gathers web insights beyond Amazon using Serper API
  • Analyzes data to produce meaningful visualizations
  • Displays results in an interactive dashboard
  • Compares products with web sentiment analysis
